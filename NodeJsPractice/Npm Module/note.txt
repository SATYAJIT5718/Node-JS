How to use NPM packages in node.js?
NPM - Node Package Manager / Module

1. Chalk :- For color coding for text 
installation :- npm i  Chalk
usage : 

const chalk = require("chalk");
chalk.blue("I am Chalk") // text color will be BLUE
chalk.blue.italic("I am Chalk") // text color will be BLUE with italic
chalk.blue.underline("I am Chalk") // text color will be BLUE with underline
chalk.blue.inverse("I am Chalk") // blue background color with new text color
