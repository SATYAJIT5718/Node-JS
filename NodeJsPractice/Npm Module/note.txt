How to use NPM packages in node.js?
NPM - Node Package Manager / Module

1. Chalk :- For color coding for text 
installation :- npm i  Chalk
usage : 

const chalk = require("chalk");
chalk.blue("I am Chalk") // text color will be BLUE
chalk.blue.italic("I am Chalk") // text color will be BLUE with italic
chalk.blue.underline("I am Chalk") // text color will be BLUE with underline
chalk.blue.inverse("I am Chalk") // blue background color with new text color


2.Validator :- For validate data( return true/false boolen value)
Installation :- npm i validator
Usage:-
const EmailId = "satya@gmail.com";
const emailValidator = validator.isEmail(EmailId);
console.log("emailValidator"); o/p - true

4.Nodemon :- nodemon is a tool that helps develop Node.js based applications by automatically restarting the node application when file changes in the directory are detected.
Installation:- npm i nodemon -g(install it globally so we can use it all project)
check version:- nodemon -v
usage:- nodemon index.js(to automatically run the server on changes on file)
exit: control + c
